# 👾 Space Invaders

## About
A classic arcade Space Invaders clone built from scratch. In this project, the game features an extra boss battle that occurs every four rounds. The boss is faster, has more lives, and provides a scaling challenge for the player. 

Developed as part of an Advanced Programming curriculum, the core focus of this project was applying robust software architecture and clean code principles rather than just making a functional game.

*   **Abstract Factory Pattern:** Used to dynamically generate enemy waves and scale difficulty without hardcoding specific alien types into the main game loop.
*   **Polymorphism:** All game entities (Player, Aliens, Projectiles) inherit from a unified base class, allowing the game engine to efficiently update and render objects at runtime without repetitive code.

## Tech Stack
*   **Language:** Java
*   **Paradigm:** Object-Oriented Programming (OOP)

## How to Run
1. Clone this repository to your local machine.
2. Open the project in your preferred Java IDE (e.g., IntelliJ IDEA).
3. Compile and run the main application class to start the game.
